# نظام إدارة ورشة صيانة أجهزة (Tech Repair Shop System) #

1- وصف المشروع

هذا مشروع عبارة عن تيرمنال او واجهة مستخدم لمركز صيانة هواتف او لابتوب هاردوير او سوفتوير قائم علي لغة بايثون 

يسمح النظام للمستخدم :
- عرض خدمات الصيانة المتاحة
- اختيار نوع الخدمة (هاردوير أو سوفتوير)
- إضافة خدمات إلى الفاتورة
- عرض الفاتورة الحالية
- حساب الفاتورة النهائية تلقائيًا

---

2- مميزات النظام

__انواع الخدمات__ 
- خدمات الهاردوير:
  - تغيير شاشة
 - تغيير بطارية
- خدمات السوفتوير:
  - إزالة فيروسات
 - تثبيت نظام تشغيل

---

3- نظام الفاتورة
- يمكن إضافة أكثر من خدمة في فاتورة واحدة
- عرض الخدمات المختارة مع السعر
- حساب السعر النهائي تلقائيًا

---

4- طريقة الحساب
- خدمات الهاردوير:
  - سعر العمالة + سعر القطعة + 10% ضريبة
- خدمات السوفتوير:
  - سعر العمالة + رسوم ثابتة

---

5- حماية البيانات
- استخدام البرمجة الكائنية (OOP)
- حماية بيانات الخدمة باستخدام encapsulation
- عدم السماح بتعديل الأسعار مباشرة من خارج الكلاس

---

6- واجهة المستخدم
- نظام يعمل بالكامل في التيرمنال
- قائمة اختيار بسيطة
- التعامل مع الأخطاء بدون توقف البرنامج

---
